## Installation

### [Homebrew](https://brew.sh/)

```shell
brew tap raxigan/tap
brew install pcfy-my-mac
```

### Script
Requires special terminal permissions
```shell
curl -s https://raw.githubusercontent.com/raxigan/pcfy-my-mac/main/install.sh | sudo bash
```
